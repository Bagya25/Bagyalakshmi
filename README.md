# Astrology Prediction Project

## Description
This is a simple front-end astrology prediction project built using HTML, CSS, and JavaScript.
Users can enter their name and zodiac sign to receive a basic astrology prediction.

## Features
- Stylish and responsive UI
- Zodiac-based predictions
- Beginner-friendly JavaScript logic

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the project
2. Open `index.html` in a browser
3. Enter your name and zodiac sign
4. Click "Get Prediction"

## Future Enhancements
- Add date of birth
